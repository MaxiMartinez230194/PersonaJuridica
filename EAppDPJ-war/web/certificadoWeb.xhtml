El servicio de certificados vía web tiene como objetivo ofrecer a las Entidades registradas la posibilidad de obtener un certificado en el cual se consta que la misma se encuentra vigente e inscripta en la Dirección de Personas Juridicas de La Provincia de Misiones.

Para poder utilizar este servicio es necesario que la Entidad tenga registrado un correo electrónico válido. En caso de que no lo tenga puede solicitar su registro completando y presentando el siguiente FORMULARIO.
El Certificado se emite con un Código de Seguridad, el cual se utiliza para verificar la validez del mismo.


Importante: Para que el certificado sea enviado a su correo electrónico es necesario tener pagadas las tasas de todo trámite y certificaciones. El impacto del pago de las tasas se hará pasadas las 24 horas hábiles.
